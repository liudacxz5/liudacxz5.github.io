Docker Compose 部署 RabbitMQ

1. 创建 docker-compose.yml 文件

2. 启动服务

3. 启用管理页面（如果镜像非 management 版需手动启用）

4. 访问管理界面

  • 地址: http://localhost:15672
  • 用户名: rabbit
  • 密码: 123456

关键配置说明

配置项 说明
image: rabbitmq:management 官方镜像（已包含管理插件）
ports 5672: AMQP 协议端口, 15672: Web 管理端
volumes Windows 路径需替换为实际目录（Linux/Mac 示例:./data:/var/lib/rabbitmq）
restart: always 容器意外退出时自动重启

注意：

  • 若使用非 Windows 系统，请将 volumes 路径改为 Linux/Mac 格式（如 /opt/rabbitmq/data:/var/lib/rabbitmq）
  • 首次启动后管理界面可能需 20-30 秒初始化
